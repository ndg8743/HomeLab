Terraform Directory Structure
================================================================================

/root/traefik-setup/terraform/
│
├── Core Configuration
│   ├── provider.tf                 # Docker provider configuration
│   ├── variables.tf                # All input variables with defaults
│   ├── outputs.tf                  # Output values for infrastructure info
│   └── terraform.tfvars.example    # Example configuration file
│
├── Infrastructure Resources
│   ├── network.tf                  # Docker network definition
│   ├── traefik.tf                  # Traefik reverse proxy container
│   ├── portainer.tf                # Portainer management UI container
│   └── personal-site.tf            # PersonalSite application container
│
├── Documentation
│   ├── README.md                   # Comprehensive documentation
│   ├── QUICKSTART.md               # Quick start guide
│   └── STRUCTURE.txt               # This file
│
└── Version Control
    └── .gitignore                  # Git ignore patterns

Generated by Terraform (not in version control):
    ├── .terraform/                 # Provider plugins and modules
    ├── .terraform.lock.hcl         # Provider version lock file
    ├── terraform.tfstate           # Current infrastructure state
    └── terraform.tfstate.backup    # Previous state backup

User Configuration (gitignored):
    └── terraform.tfvars            # Your custom variable values

================================================================================
